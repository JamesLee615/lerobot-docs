关键设计决策：

  1. nobssteps=1：ACT 只使用当前帧的观察，不使用历史帧。这意味着 observationdeltaindices 返回 None。
  1. ndecoderlayers=1：尽管原始论文使用了 7 层 Decoder，但由于 bug，实际上只有第一层有梯度更新。LeRobot 保留了原始行为。
  1. chunksize=100 且 nactionsteps=100：ACT 一次性生成 100 步动作并全部执行。没有像 DP 那样的 "生成 horizon 步但只执行 naction_steps 步" 的机制。
  1. 不使用 framestride：ACT 处理每一帧，而 DP 使用 framestride=3 在 30Hz 数据上进行子采样训练在 10Hz。

2.3.1 select_action — 动作选择

@torch.no_grad()
def select_action(self, batch: dict[str, Tensor]) -> Tensor:
    self.eval()

    # 模式 1: 时序集成 (temporal ensembling)
    if self.config.temporal_ensemble_coeff is not None:
        actions = self.predict_action_chunk(batch)          # (B, chunk_size, action_dim)
        action = self.temporal_ensembler.update(actions)     # 加权平均
        return action

    # 模式 2: 动作队列 (默认)
    if len(self._action_queue) == 0:
        # 队列为空时，从模型获取新动作 chunk
        actions = self.predict_action_chunk(batch)
        actions = actions[:, :self.config.n_action_steps]
        # 转置后入队: (chunk_size, B, action_dim)
        self._action_queue.extend(actions.transpose(0, 1))
    return self._action_queue.popleft()

时序集成机制 (当 temporalensemblecoeff != None)：

  • 每个时间步，模型预测一个 chunk_size 的动作序列
  • 使用指数加权移动平均：w_i = exp(-coeff * i)，旧动作权重更高（coeff=0.01 默认）
  • 这是原始 ACT 论文的 Algorithm 2，用于平滑跨时间步的动作预测

2.3.2 predictactionchunk — 预测动作块

@torch.no_grad()
def predict_action_chunk(self, batch: dict[str, Tensor]) -> Tensor:
    self.eval()
    # 将分散的图像键合并为 OBS_IMAGES 列表
    if self.config.image_features:
        batch = dict(batch)
        batch[OBS_IMAGES] = [batch[key] for key in self.config.image_features]
    actions = self.model(batch)[0]    # 只取动作，不要 (mu, log_sigma)
    return actions                     # (B, chunk_size, action_dim)

2.3.3 forward — 训练前向传播

def forward(self, batch: dict[str, Tensor]) -> tuple[Tensor, dict]:
    if self.config.image_features:
        batch = dict(batch)
        batch[OBS_IMAGES] = [batch[key] for key in self.config.image_features]

    # 模型前向：返回动作和 VAE 潜在参数
    actions_hat, (mu_hat, log_sigma_x2_hat) = self.model(batch)

    # L1 重建损失，忽略 padding 位置
    l1_loss = (
        F.l1_loss(batch[ACTION], actions_hat, reduction="none")
        * ~batch["action_is_pad"].unsqueeze(-1)
    ).mean()

    loss_dict = {"l1_loss": l1_loss.item()}

    if self.config.use_vae:
        # KL 散度: D_KL(q(z|x) || p(z))，p(z) = N(0, I)
        # 对每个潜在维度独立求和，再对 batch 取平均
        mean_kld = (
            (-0.5 * (1 + log_sigma_x2_hat - mu_hat.pow(2)
                     - (log_sigma_x2_hat).exp()))
            .sum(-1).mean()
        )
        loss_dict["kld_loss"] = mean_kld.item()
        loss = l1_loss + mean_kld * self.config.kl_weight  # kl_weight=10.0
    else:
        loss = l1_loss

    return loss, loss_dict

损失函数：L = L1(actions_hat, actions) + 10.0 × KL(N(μ,σ²) || N(0,I))

2.4.2 init 详细代码路径

class ACT(nn.Module):
    def __init__(self, config: ACTConfig):
        super().__init__()
        self.config = config

        # === 1. VAE Encoder (可选) ===
        if self.config.use_vae:
            # BERT-style VAE encoder: [cls_token, *robot_state, *action_sequence]
            self.vae_encoder = ACTEncoder(config, is_vae_encoder=True)
            self.vae_encoder_cls_embed = nn.Embedding(1, config.dim_model)

            # 输入投影
            if self.config.robot_state_feature:
                self.vae_encoder_robot_state_input_proj = nn.Linear(
                    state_dim, config.dim_model
                )
            self.vae_encoder_action_input_proj = nn.Linear(
                action_dim, config.dim_model
            )

            # 潜在空间投影: 输出 mean 和 log_variance
            self.vae_encoder_latent_output_proj = nn.Linear(
                config.dim_model, config.latent_dim * 2
            )

            # 固定正弦位置编码
            num_input_tokens = 1 + (1 if robot_state else 0) + config.chunk_size
            self.register_buffer(
                "vae_encoder_pos_enc",
                create_sinusoidal_pos_embedding(num_input_tokens, dim_model)
            )

        # === 2. 视觉骨干网络 ===
        if self.config.image_features:
            backbone_model = getattr(torchvision.models, config.vision_backbone)(
                replace_stride_with_dilation=[False, False, config.replace_final_stride_with_dilation],
                weights=config.pretrained_backbone_weights,
                norm_layer=FrozenBatchNorm2d,   # 固定 BatchNorm 参数
            )
            self.backbone = IntermediateLayerGetter(
                backbone_model, return_layers={"layer4": "feature_map"}
            )

        # === 3. 触觉编码器 ===
        if self.config.use_tactile:
            self.tactile_encoder = TactileTokenEncoder(
                encoder_type=config.tactile_encoder_type,
                input_shape=config.tactile_input_shape,
                feature_dim=config.dim_model,     # 直接输出 dim_model 维度
                n_tokens=config.n_tactile_tokens,
                dropout=config.tactile_dropout,
            )

        # === 4. Transformer Encoder + Decoder ===
        self.encoder = ACTEncoder(config)     # 4 层 TransformerEncoder
        self.decoder = ACTDecoder(config)     # 1 层 TransformerDecoder

        # Encoder 输入投影
        self.encoder_latent_input_proj = nn.Linear(latent_dim, dim_model)
        if robot_state:
            self.encoder_robot_state_input_proj = nn.Linear(state_dim, dim_model)
        if env_state:
            self.encoder_env_state_input_proj = nn.Linear(env_dim, dim_model)
        if image_features:
            self.encoder_img_feat_input_proj = nn.Conv2d(
                backbone_fc_in_features, dim_model, kernel_size=1
            )

        # 位置编码
        # 1D tokens: [latent, robot_state, env_state, tactile_tokens...]
        n_1d_tokens = 1 + has_robot_state + has_env_state + n_tactile_tokens
        self.encoder_1d_feature_pos_embed = nn.Embedding(n_1d_tokens, dim_model)
        # 2D 空间编码用于图像特征图的每个像素
        if image_features:
            self.encoder_cam_feat_pos_embed = ACTSinusoidalPositionEmbedding2d(
                dim_model // 2
            )

        # Decoder 位置编码
        self.decoder_pos_embed = nn.Embedding(chunk_size, dim_model)

        # === 5. 动作输出头 ===
        self.action_head = nn.Linear(dim_model, action_dim)

        self._reset_parameters()  # Xavier uniform 初始化

2.4.3 forward 详细数据流

def forward(self, batch) -> tuple[Tensor, tuple[Tensor, Tensor]]:
    """
    输入 batch:
      OBS_STATE:   (B, state_dim)         # 当前关节角度
      OBS_IMAGES:  list[(B, C, H, W)]     # 每张图像单独处理
      OBS_TACTILE: (B, H_tact, W_tact)     # 触觉传感器数据
      ACTION:      (B, chunk_size, action_dim)  # 仅在训练时存在
      action_is_pad: (B, chunk_size)       # padding 掩码

    输出:
      actions:  (B, chunk_size, action_dim)
      (mu, log_sigma_x2): VAE 潜在参数
    """

    # === 第1步: VAE 编码 (仅训练时) ===
    if use_vae and ACTION in batch and self.training:
        # 组装 VAE encoder 输入序列
        cls_embed = repeat(vae_encoder_cls_embed, "1 d -> b 1 d", b=B)
        robot_state_embed = vae_encoder_robot_state_input_proj(OBS_STATE).unsqueeze(1)
        action_embed = vae_encoder_action_input_proj(ACTION)

        vae_input = torch.cat([cls_embed, robot_state_embed, action_embed], dim=1)
        # vae_input: (B, 1+1+chunk_size, dim_model)

        # 添加 key_padding_mask (标记填充的 action)
        key_padding_mask = torch.cat([
            torch.full((B, 2), False),      # CLS 和 robot_state 都不是 padding
            batch["action_is_pad"]           # action 的 padding 掩码
        ], dim=1)

        # VAE Encoder 前向
        cls_token_out = self.vae_encoder(
            vae_input.permute(1, 0, 2),      # (seq, B, dim)
            pos_embed=vae_encoder_pos_enc.permute(1, 0, 2),
            key_padding_mask=key_padding_mask,
        )[0]  # 取 CLS token: (B, dim_model)

        # 投影到潜在分布参数
        latent_pdf = vae_encoder_latent_output_proj(cls_token_out)
        mu = latent_pdf[:, :latent_dim]
        log_sigma_x2 = latent_pdf[:, latent_dim:]  # 实际是 2*log(σ)

        # 重参数化采样
        latent_sample = mu + (log_sigma_x2 / 2).exp() * randn_like(mu)
    else:
        # 推理时 latent_sample 设为全零
        mu = log_sigma_x2 = None
        latent_sample = zeros(B, latent_dim)  # 无梯度

    # === 第2步: 组装 Transformer Encoder 输入 ===
    encoder_in_tokens = [encoder_latent_input_proj(latent_sample)]
    encoder_in_pos_embed = list(encoder_1d_feature_pos_embed.weight.unsqueeze(1))

    # 添加 robot_state token
    if robot_state:
        encoder_in_tokens.append(encoder_robot_state_input_proj(OBS_STATE))
    # 添加 env_state token
    if env_state:
        encoder_in_tokens.append(encoder_env_state_input_proj(OBS_ENV_STATE))
    # 添加触觉 tokens
    if use_tactile:
        for tactile_key in tactile_keys:
            tactile_tokens = self.tactile_encoder(batch[tactile_key])
            # (B, n_tokens, dim_model)
            for i in range(n_tactile_tokens):
                encoder_in_tokens.append(tactile_tokens[:, i])

    # 添加图像特征 (展平为序列)
    if image_features:
        for img in OBS_IMAGES:
            # ResNet 特征提取
            cam_features = self.backbone(img)["feature_map"]
            # (B, C_feat, H_feat, W_feat)

            # 2D 正弦位置编码
            cam_pos_embed = self.encoder_cam_feat_pos_embed(cam_features)

            # 1x1 卷积投影到 dim_model
            cam_features = self.encoder_img_feat_input_proj(cam_features)

            # 展平空间维度: (B, dim_model, H_feat, W_feat) → (H*W, B, dim_model)
            cam_features = einops.rearrange(cam_features, "b c h w -> (h w) b c")
            cam_pos_embed = einops.rearrange(cam_pos_embed, "b c h w -> (h w) b c")

            encoder_in_tokens.extend(list(cam_features))
            encoder_in_pos_embed.extend(list(cam_pos_embed))

    # 堆叠: (total_tokens, B, dim_model)
    encoder_in_tokens = torch.stack(encoder_in_tokens, dim=0)
    encoder_in_pos_embed = torch.stack(encoder_in_pos_embed, dim=0)

    # === 第3步: Transformer Encoder ===
    encoder_out = self.encoder(encoder_in_tokens, pos_embed=encoder_in_pos_embed)
    # (total_tokens, B, dim_model)

    # === 第4步: Transformer Decoder ===
    # Decoder 输入: 全零 tokens × chunk_size
    decoder_in = torch.zeros(chunk_size, B, dim_model)
    decoder_out = self.decoder(
        decoder_in,
        encoder_out,                                          # cross-attention memory
        encoder_pos_embed=encoder_in_pos_embed,
        decoder_pos_embed=self.decoder_pos_embed.weight.unsqueeze(1),
    )
    # (chunk_size, B, dim_model)

    # === 第5步: 动作预测 ===
    decoder_out = decoder_out.transpose(0, 1)  # (B, chunk_size, dim_model)
    actions = self.action_head(decoder_out)     # (B, chunk_size, action_dim)

    return actions, (mu, log_sigma_x2)

2.4.4 Transformer Encoder/Decoder 内部实现

ACTEncoder: 多层 nn.TransformerEncoderLayer 的堆叠

class ACTEncoder(nn.Module):
    def __init__(self, config, is_vae_encoder=False):
        n_layers = config.n_vae_encoder_layers if is_vae_encoder else config.n_encoder_layers
        # 创建 n_layers 个 TransformerEncoderLayer
        encoder_layer = nn.TransformerEncoderLayer(
            d_model=config.dim_model,
            nhead=config.n_heads,
            dim_feedforward=config.dim_feedforward,
            dropout=config.dropout,
            activation=config.feedforward_activation,
            norm_first=config.pre_norm,  # False: post-norm
            batch_first=False,            # (seq, batch, feature)
        )
        self.layers = nn.ModuleList([copy.deepcopy(encoder_layer) for _ in range(n_layers)])

    def forward(self, src, pos_embed=None, key_padding_mask=None):
        if pos_embed is not None:
            src = src + pos_embed
        output = src
        for layer in self.layers:
            output = layer(output, src_key_padding_mask=key_padding_mask)
        return output

ACTDecoder: nn.TransformerDecoder

class ACTDecoder(nn.Module):
    def __init__(self, config):
        decoder_layer = nn.TransformerDecoderLayer(
            d_model=config.dim_model,
            nhead=config.n_heads,
            dim_feedforward=config.dim_feedforward,
            dropout=config.dropout,
            activation=config.feedforward_activation,
            norm_first=config.pre_norm,
            batch_first=False,
        )
        self.layers = nn.ModuleList(
            [copy.deepcopy(decoder_layer) for _ in range(config.n_decoder_layers)]
        )

    def forward(self, tgt, memory, encoder_pos_embed, decoder_pos_embed):
        tgt = tgt + decoder_pos_embed
        output = tgt
        for layer in self.layers:
            output = layer(
                output,
                memory,
                memory_key_padding_mask=None,
            )
        return output

关键细节：

  • Encoder 和 VAE Encoder 共享相同的 ACTEncoder 类，通过 isvaeencoder 参数区分层数
  • Decoder 使用 batch_first=False（PyTorch 默认的 (seq, batch, feature) 格式）
  • FrozenBatchNorm2d：在训练时冻结 BatchNorm 统计量以保留预训练特征

2.5 时序集成器: ACTTemporalEnsembler

文件: src/lerobot/policies/act/modeling_act.py:166-254

class ACTTemporalEnsembler:
    def __init__(self, temporal_ensemble_coeff: float, chunk_size: int):
        self.chunk_size = chunk_size
        # 计算指数权重: w_i = exp(-coeff * i)
        self.ensemble_weights = torch.exp(
            -temporal_ensemble_coeff * torch.arange(chunk_size)
        )
        self.ensemble_weights_cumsum = torch.cumsum(self.ensemble_weights, dim=0)
        self.reset()

    def reset(self):
        self.ensembled_actions = None          # (B, chunk_size, action_dim)
        self.ensembled_actions_count = None     # (chunk_size, 1) 计数

    def update(self, actions: Tensor) -> Tensor:
        """
        actions: (B, chunk_size, action_dim) 新预测的动作 chunk

        返回: (B, action_dim) 当前时间步的集成动作
        """
        if self.ensembled_actions is None:
            # 第一步：直接用新预测初始化
            self.ensembled_actions = actions.clone()
            self.ensembled_actions_count = torch.ones(
                (self.chunk_size, 1), dtype=torch.long
            )
        else:
            # 在线更新: 对前 chunk_size-1 个动作进行加权平均
            # 旧动作权重 = cumsum[count-1]
            # 新动作权重 = weights[count]
            self.ensembled_actions *= self.ensemble_weights_cumsum[
                self.ensembled_actions_count - 1
            ]
            self.ensembled_actions += (
                actions[:, :-1] * self.ensemble_weights[self.ensembled_actions_count]
            )
            self.ensembled_actions /= self.ensemble_weights_cumsum[
                self.ensembled_actions_count
            ]
            self.ensembled_actions_count = torch.clamp(
                self.ensembled_actions_count + 1, max=self.chunk_size
            )
            # 最后一个动作（无历史）直接追加
            self.ensembled_actions = torch.cat(
                [self.ensembled_actions, actions[:, -1:]], dim=1
            )
            self.ensembled_actions_count = torch.cat([
                self.ensembled_actions_count,
                torch.ones_like(self.ensembled_actions_count[-1:])
            ])

        # 消费第一个动作
        action = self.ensembled_actions[:, 0]
        self.ensembled_actions = self.ensembled_actions[:, 1:]
        self.ensembled_actions_count = self.ensembled_actions_count[1:]
        return action

时序集成的数学原理：

对于时间步 t 预测的动作序列的第 i 步动作（i=0 表示最旧），使用指数衰减权重：

$$w_i = e^{-\beta \cdot i}$$

最终该步的集成动作为所有跨时间步预测的加权平均。在原始 ACT 实现中，β=0.01，这意味着旧的预测获得更高的权重。

3.3.2 select_action — 动作选择

@torch.no_grad()
def select_action(self, batch: dict[str, Tensor], noise=None) -> Tensor:
    """
    时序示意图 (n_obs_steps=2, horizon=16, n_action_steps=8):

    timestep:        | t-1  | t    | t+1  | ... | t+7  | t+8  | ... | t+14 |
    observation:     | YES  | YES  | NO   | ... | NO   | NO   | ... | NO   |
    action generated:| YES  | YES  | YES  | ... | YES  | YES  | ... | YES  |
    action executed: | NO   | YES  | YES  | ... | YES  | NO   | ... | NO   |
    """
    if ACTION in batch:
        batch.pop(ACTION)

    # 将多相机图像堆叠为 (B, num_cameras, C, H, W)
    if image_features:
        batch = dict(batch)
        batch[OBS_IMAGES] = torch.stack(
            [batch[key] for key in image_features], dim=-4
        )

    # 更新队列 (自动填充)
    self._queues = populate_queues(self._queues, batch)

    # 动作队列为空 → 从模型生成新动作
    if len(self._queues[ACTION]) == 0:
        actions = self.predict_action_chunk(batch, noise=noise)
        # actions: (B, n_action_steps, action_dim)
        self._queues[ACTION].extend(actions.transpose(0, 1))

    return self._queues[ACTION].popleft()

populate_queues 函数 (src/lerobot/policies/utils.py):

def populate_queues(queues, batch):
    for key in batch:
        if key not in queues:
            continue
        # 对每个 batch 元素独立处理
        for i in range(len(batch[key])):
            queues[key].append(batch[key][i])
    return queues

队列机制详解：

  1. 每次调用 selectaction，当前观察被推入对应的观察队列（长度= nobs_steps）
  2. 当动作队列为空时，从观察队列中取最近 nobssteps 帧 → 堆叠 → 去噪生成 horizon 步动作 → 取前 nactionsteps 步入队
  3. 从动作队列弹出一个动作执行
  4. 这保证了在固定的观察历史下进行推理，同时避免每步都运行扩散去噪

3.3.3 predictactionchunk — 预测

@torch.no_grad()
def predict_action_chunk(self, batch, noise=None) -> Tensor:
    # 从队列堆叠观察历史
    batch = {
        k: torch.stack(list(self._queues[k]), dim=1)
        for k in batch if k in self._queues
    }
    # batch[OBS_STATE]: (B, n_obs_steps, state_dim)
    # batch[OBS_IMAGES]: (B, n_obs_steps, num_cameras, C, H, W)

    actions = self.diffusion.generate_actions(batch, noise=noise)
    return actions  # (B, n_action_steps, action_dim)

3.4.2 训练: compute_loss

def compute_loss(self, batch) -> Tensor:
    """
    batch:
      OBS_STATE:    (B, n_obs_steps, state_dim)
      OBS_IMAGES:   (B, n_obs_steps, num_cameras, C, H, W)
      ACTION:       (B, horizon, action_dim)
      action_is_pad: (B, horizon)
    """
    # === 第1步: 编码全局条件 ===
    global_cond = self._prepare_global_conditioning(batch)
    # (B, global_cond_dim × n_obs_steps)

    # === 第2步: 前向扩散 ===
    trajectory = batch[ACTION]  # (B, horizon, action_dim)
    eps = torch.randn(trajectory.shape)  # 采样噪声

    # 为 batch 中每个样本采样不同的扩散时间步
    timesteps = torch.randint(
        0, num_train_timesteps, size=(B,)
    ).long()

    # 加噪: x_t = √(α̅_t) × x_0 + √(1-α̅_t) × ε
    noisy_trajectory = self.noise_scheduler.add_noise(
        trajectory, eps, timesteps
    )

    # === 第3步: U-Net 预测 ===
    pred = self.unet(noisy_trajectory, timesteps, global_cond=global_cond)

    # === 第4步: 损失计算 ===
    if prediction_type == "epsilon":
        target = eps              # 预测噪声
    elif prediction_type == "sample":
        target = trajectory       # 预测原始样本

    loss = F.mse_loss(pred, target, reduction="none")

    # 可选: 填充位置损失遮罩
    if do_mask_loss_for_padding:
        loss = loss * ~batch["action_is_pad"].unsqueeze(-1)

    return loss.mean()

3.4.3 推理: generateactions + conditionalsample

def generate_actions(self, batch, noise=None) -> Tensor:
    """
    从观察编码全局条件 → 从噪声去噪 → 截取 n_action_steps 动作
    """
    # 编码全局条件
    global_cond = self._prepare_global_conditioning(batch)

    # 扩散去噪采样
    actions = self.conditional_sample(
        batch_size, global_cond=global_cond, noise=noise
    )
    # (B, horizon, action_dim)

    # 截取 n_action_steps 动作 (从当前观察开始)
    start = n_obs_steps - 1     # = 1
    end = start + n_action_steps # = 1 + 8 = 9
    actions = actions[:, start:end]
    # (B, n_action_steps, action_dim)

    return actions

def conditional_sample(self, batch_size, global_cond, generator, noise):
    """迭代去噪过程 (DDPM/DDIM)"""

    # 初始化纯噪声
    sample = noise if noise is not None else torch.randn(
        (batch_size, horizon, action_dim)
    )

    self.noise_scheduler.set_timesteps(self.num_inference_steps)

    for t in self.noise_scheduler.timesteps:
        # 预测噪声 (或样本)
        model_output = self.unet(
            sample,
            torch.full((batch_size,), t, dtype=torch.long),
            global_cond=global_cond,
        )
        # 一步去噪: x_{t-1} = DDPM/DDIM step
        sample = self.noise_scheduler.step(
            model_output, t, sample, generator=generator
        ).prev_sample

    return sample

3.4.4 全局条件编码: prepareglobal_conditioning

def _prepare_global_conditioning(self, batch) -> Tensor:
    """
    编码所有模态 → 拼接 → 展平为 (B, global_cond_dim × n_obs_steps)
    """
    global_cond_feats = []

    # (1) 机器人状态
    if robot_state:
        global_cond_feats.append(batch[OBS_STATE])  # (B, n_obs_steps, state_dim)

    # (2) 图像特征
    if image_features:
        if use_separate_rgb_encoder_per_camera:
            # 每个相机独立编码
            images_per_camera = einops.rearrange(
                batch[OBS_IMAGES], "b s n ... -> n (b s) ..."
            )
            img_features_list = torch.cat([
                encoder(images)
                for encoder, images in zip(rgb_encoder, images_per_camera, strict=True)
            ])
            img_features = einops.rearrange(
                img_features_list, "(n b s) ... -> b s (n ...)",
                b=B, s=n_obs_steps
            )
        else:
            # 共享编码器
            img_features = self.rgb_encoder(
                einops.rearrange(batch[OBS_IMAGES], "b s n ... -> (b s n) ...")
            )
            img_features = einops.rearrange(
                img_features, "(b s n) ... -> b s (n ...)",
                b=B, s=n_obs_steps
            )
        global_cond_feats.append(img_features)
        # img_features: (B, n_obs_steps, 64 × num_cameras)

    # (3) 环境状态
    if env_state:
        global_cond_feats.append(batch[OBS_ENV_STATE])

    # (4) 触觉特征
    if use_tactile:
        for tactile_key in tactile_keys:
            tactile_data = batch[tactile_key]  # (B, n_obs_steps, H, W)
            bs, n_obs = tactile_data.shape[:2]
            tactile_flat = tactile_data.flatten(0, 1)  # (B*n_obs, H, W)
            chunks = self.tactile_encoder(tactile_flat)
            # (B*n_obs, n_chunks, feat_dim)
            chunks = chunks.flatten(1)  # (B*n_obs, n_chunks*feat_dim)
            chunks = chunks.view(bs, n_obs, -1)
            # (B, n_obs_steps, n_chunks*feat_dim)
            global_cond_feats.append(chunks)

    # 拼接所有特征 → 展平时序维度
    return torch.cat(global_cond_feats, dim=-1).flatten(start_dim=1)
    # (B, sum_all_dims × n_obs_steps)

class DiffusionRgbEncoder(nn.Module):
    def __init__(self, config: DiffusionConfig):
        super().__init__()

        # 图像预处理
        if config.resize_shape is not None:
            self.resize = torchvision.transforms.Resize(config.resize_shape)
        if config.crop_shape is not None:
            self.do_crop = True
            self.center_crop = torchvision.transforms.CenterCrop(crop_shape)
            self.maybe_random_crop = (
                torchvision.transforms.RandomCrop(crop_shape)
                if config.crop_is_random else self.center_crop
            )

        # ResNet 骨干 (去掉最后两层: avgpool + fc)
        backbone_model = getattr(torchvision.models, config.vision_backbone)(
            weights=config.pretrained_backbone_weights
        )
        self.backbone = nn.Sequential(*list(backbone_model.children())[:-2])

        # 可选: BatchNorm → GroupNorm (适合小 batch size)
        if config.use_group_norm:
            self.backbone = _replace_submodules(
                root_module=self.backbone,
                predicate=lambda x: isinstance(x, nn.BatchNorm2d),
                func=lambda x: nn.GroupNorm(
                    num_groups=x.num_features // 16,
                    num_channels=x.num_features
                ),
            )

        # Spatial Softmax 池化 (关键设计)
        dummy_shape = compute_feature_map_shape()
        self.pool = SpatialSoftmax(
            dummy_shape[1:],  # (C_feat, H_feat, W_feat)
            num_kp=config.spatial_softmax_num_keypoints  # 32
        )

        # 输出投影
        self.feature_dim = config.spatial_softmax_num_keypoints * 2  # 64
        self.out = nn.Linear(
            config.spatial_softmax_num_keypoints * 2, self.feature_dim
        )

    def forward(self, x: Tensor) -> Tensor:
        """
        x: (B, C, H, W)  RGB 图像

        Returns: (B, feature_dim)  = (B, 64)
        """
        if self.resize:
            x = self.resize(x)
        if self.do_crop:
            if self.training:
                x = self.maybe_random_crop(x)
            else:
                x = self.center_crop(x)

        # ResNet 提取特征图
        x = self.backbone(x)  # (B, C_feat, H_feat, W_feat)

        # Spatial Softmax → 关键点坐标
        x = self.pool(x)      # (B, 32, 2)  # 32个关键点, 每个有(x,y)坐标

        x = x.flatten(1)      # (B, 64)
        x = self.out(x)       # (B, 64)  # 最终特征
        return x

Spatial Softmax 详解

Spatial Softmax 是 Diffusion Policy 的关键组件：

class SpatialSoftmax(nn.Module):
    """
    从 2D 特征图提取"激活中心"作为空间关键点

    输入: (B, C, H, W) 特征图
    输出: (B, K, 2) K 个关键点的 (x,y) 归一化坐标 (范围 [-1, 1])

    步骤:
    1. 可选: 1x1 卷积调整通道数 (当 K != C 时)
    2. 对空间维度做 softmax (每个通道独立)
    3. 计算期望坐标 = softmax_weights @ pos_grid
    """
    def __init__(self, input_shape, num_kp=None):
        self._in_c, self._in_h, self._in_w = input_shape
        if num_kp is not None:
            self.nets = nn.Conv2d(self._in_c, num_kp, kernel_size=1)
            self._out_c = num_kp

        # 预计算归一化坐标网格 (-1 到 1)
        pos_x, pos_y = np.meshgrid(
            np.linspace(-1, 1, self._in_w),
            np.linspace(-1, 1, self._in_h)
        )
        self.register_buffer("pos_grid",
            torch.cat([pos_x.reshape(-1,1), pos_y.reshape(-1,1)], dim=1)
        )  # (H*W, 2)

    def forward(self, features: Tensor) -> Tensor:
        if self.nets is not None:
            features = self.nets(features)  # (B, C_out, H, W)

        # Softmax 归一化
        features = features.reshape(-1, H * W)   # (B*K, H*W)
        attention = F.softmax(features, dim=-1)   # 每个通道的 attention 和为1

        # 期望坐标: attention @ pos_grid
        expected_xy = attention @ self.pos_grid   # (B*K, 2)

        return expected_xy.view(-1, self._out_c, 2)  # (B, K, 2)

架构：带有 FiLM 条件调制的 1D 卷积 U-Net

输入: noisy_trajectory (B, horizon, action_dim)
       timestep (B,) + global_cond (B, cond_dim)

编码器(下采样):
  DownBlock1: Conv1d → FiLM → Conv1d → FiLM → Downsample(stride=2)
  DownBlock2: Conv1d → FiLM → Conv1d → FiLM → Downsample(stride=2)
  DownBlock3: Conv1d → FiLM → Conv1d → FiLM → Downsample(stride=2)

瓶颈:
  MiddleBlock: Conv1d → FiLM → Conv1d → FiLM

解码器(上采样):
  UpBlock3: Upsample → Conv1d → FiLM → Conv1d → FiLM
  UpBlock2: Upsample → Conv1d → FiLM → Conv1d → FiLM
  UpBlock1: Upsample → Conv1d → FiLM → Conv1d → FiLM

最终: Conv1d(1×1) → (B, horizon, action_dim)

FiLM 调制细节：

# 每个残差块内的 FiLM:
emb = concat([timestep_embedding(t), global_cond])  # 拼接时间和全局条件
emb = Linear(emb) → Mish → Linear(emb)  # 投影到 scale + bias

# if use_film_scale_modulation:
out = conv(out) * (1 + scale) + bias  # 同时调制 scale 和 bias
# else:
out = conv(out) + bias  # 仅调制 bias

特征维度变化 (以 down_dims=(512, 1024, 2048) 为例)：

输入:     (B, action_dim, horizon)          6 × 16
Down1:    (B, 512, 8)                      stride=2 下采样
Down2:    (B, 1024, 4)                     stride=2
Down3:    (B, 2048, 2)                     stride=2
Middle:   (B, 2048, 2)
Up3:      (B, 1024, 4)                     skip+conv

输出:     (B, horizon, action_dim)         16 × 6

4. 触觉传感器扩展

4.1 共享触觉编码器: TactileTokenEncoder

文件: src/lerobot/policies/tactile/encoder.py

两种实现：

4.1.1 TactileCNN — 基础 CNN

class TactileCNN(nn.Module):
    """
    输入: (B, H, W) 触觉传感器读数  示例: (B, 12, 32)
    输出: (B, feature_dim)           示例: (B, 64)

    架构:
      Conv2d(1→32, k3) → BN → ReLU → MaxPool2d(2)  # (B, 32, H/2, W/2)
      Conv2d(32→64, k3) → BN → ReLU → MaxPool2d(2)  # (B, 64, H/4, W/4)
      Conv2d(64→128, k3) → BN → ReLU → MaxPool2d(2) # (B, 128, H/8, W/8)
      Flatten → Linear → ReLU → Dropout → Linear
    """

4.1.2 TactileAttentionCNN — 带空间注意力

class TactileAttentionCNN(nn.Module):
    """
    与 TactileCNN 类似，但在最后一个卷积层后添加空间注意力:
      Conv2d(256→128, k1) → ReLU → Conv2d(128→1, k1) → Sigmoid
    将注意力权重与特征图逐元素相乘
    然后使用 GlobalAvgPool + GlobalMaxPool → Concat → FC
    """

4.1.3 TactileTokenEncoder — 多 Token 包装器

class TactileTokenEncoder(nn.Module):
    def __init__(self, encoder_type, input_shape, feature_dim, n_tokens, dropout):
        self.n_tokens = n_tokens
        # 选择骨干
        if encoder_type == "cnn":
            self.backbone = TactileCNN(input_shape, feature_dim, dropout)
        elif encoder_type == "attention":
            self.backbone = TactileAttentionCNN(input_shape, feature_dim, dropout)

        # 当 n_tokens > 1 时: 投影到多个 token
        self.token_proj = (
            nn.Linear(feature_dim, n_tokens * feature_dim)
            if n_tokens > 1 else None
        )

    def forward(self, x: Tensor) -> Tensor:
        """
        x: (B, H, W)
        returns: (B, n_tokens, feature_dim)
        """
        feat = self.backbone(x)  # (B, feature_dim)
        if self.n_tokens == 1:
            return feat.unsqueeze(1)  # (B, 1, feature_dim)
        feat = self.token_proj(feat)  # (B, n_tokens * feature_dim)
        return feat.view(B, self.n_tokens, self.feature_dim)

在 DP 中，触觉特征被 展平并拼接 到全局条件向量中：

关键区别：

  • ACT：触觉 token 是有位置编码的独立序列元素，参与 Transformer 的注意力计算
  • DP：触觉特征是全局条件的组成部分，通过 FiLM 调制影响 U-Net 的每一层

6. 推理流程对比

6.1 ACT 推理

输入: 当前帧观察

1. 图像: ResNet(冻结BN) → feature_map → flatten → 位置编码 → tokens
2. 状态: Linear → token
3. 触觉: TactileTokenEncoder → token(s)
4. 潜在向量: zeros(B, 32) (推理时)
5. Transformer Encoder: 融合所有 tokens
6. Transformer Decoder: 从零 tokens 进行交叉注意力解码, 输出 chunk_size=100 动作
7. (可选) Temporal Ensemble: 对跨时间步的预测进行指数加权平均
8. 执行前 n_action_steps=100 步动作 (一次性全部执行)

速度: 极快 — 单次前向传播

6.2 Diffusion Policy 推理

输入: 最近 n_obs_steps=2 帧的观察 (从队列堆叠)

1. 图像: Resize→Crop→ResNet→SpatialSoftmax(32 kp)→Linear(64)
   每帧每个相机输出 (B, 64), 对所有相机/时间步拼接
2. 状态: 直接拼接 (B, n_obs_steps, state_dim)
3. 触觉: TactileTokenEncoder→flatten→拼接
4. 全局条件: concat([图像特征, 状态, 触觉特征]) → flatten → (B, cond_dim)
5. DDPM 去噪循环 (100步):
   for t from T down to 1:
       noise_pred = UNet(noisy_actions, t_embed, global_cond)
       noisy_actions = scheduler.step(noise_pred, t, noisy_actions)
6. 从 horizon=16 步中取第 1~8 步 (n_obs_steps-1 ~ n_obs_steps-1+n_action_steps)
7. 动作入队, 逐步执行

速度: 较慢 — 100 步迭代去噪，每步都要运行 U-Net

6.3 关键行为差异

ACT Diffusion Policy
推理频率 每 episode 可能只推理 1 次 (如果 episode ≤ 100步) 每 8 步推理一次
动作一致性 chunk 内动作平滑 (一次生成) 跨 chunk 可能不连续 (但差异小)
对噪声的鲁棒性 依赖时序集成 扩散模型固有的多峰能力
实时性 极佳 (单次前向) 可接受 (DDIM 可减少推理步数)
多模态行为 单峰 (高斯 VAE) 多峰 (扩散模型天然支持)
