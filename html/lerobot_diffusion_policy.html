1. 概述

Diffusion Policy 来自论文 Diffusion Policy: Visuomotor Policy Learning via Action Diffusion (https://huggingface.co/papers/2303.04137)。

核心思想：将机器人动作预测建模为条件扩散去噪过程——从纯噪声开始，在观察条件的引导下，通过多步迭代去噪生成连贯的动作轨迹。

特性 Diffusion Policy
动作预测方式 迭代式扩散去噪，从纯噪声逐步去噪
核心网络 ResNet 视觉编码器 + 1D 条件 U-Net
观察步数 nobssteps=2 (含历史帧)
动作 Horizon horizon=16 (16步动作预测)
执行步数 nactionsteps=8 (horizon 的一部分)
训练目标 MSE (预测噪声 ε 或原始动作)
推理机制 多步去噪循环 (DDPM/DDIM)
时序机制 队列：观察历史缓存 + 动作出队
条件注入 FiLM 调制 (扩散步嵌入)
优化器 Adam, lr=1e-4, betas=(0.95, 0.999)
学习率调度 Cosine with Warmup (500步)
帧步幅 frame_stride=3 (子采样，降低帧率)
图像预处理 Resize → RandomCrop (训练) / CenterCrop (推理)

关键设计决策

  1. dropnlastframes 自动计算：(horizon - nactionsteps - nobssteps + 1) framestride，默认 (16-8-2+1)1 = 7，丢弃 episode 末尾被 padding 的不完整帧。
  1. U-Net 下采样兼容性：downsamplingfactor = 2^len(downdims) = 8，horizon 必须是 8 的倍数。
  1. 图像预处理流程：原始图像 → Resize → RandomCrop (训练) / CenterCrop (推理)
  1. framestride 机制：在 30Hz 数据集上以 framestride=3 训练等效于在 10Hz 训练。

        config.validate_features()

4.1 reset — 初始化队列

def reset(self):
    """环境重置时调用，清空所有缓存"""
    self._queues = {
        ACTION: deque(maxlen=n_action_steps),                  # 动作队列 (长度=8)
    }
    if robot_state:
        self._queues[OBS_STATE] = deque(maxlen=n_obs_steps)    # 状态队列 (长度=2)
    if image_features:
        self._queues[OBS_IMAGES] = deque(maxlen=n_obs_steps)   # 图像队列
    if env_state:
        self._queues[OBS_ENV_STATE] = deque(maxlen=n_obs_steps)
    if use_tactile:
        for key in tactile_keys:
            self._queues[key] = deque(maxlen=n_obs_steps)       # 触觉队列

4.2 select_action — 动作选择

@torch.no_grad()
def select_action(self, batch: dict[str, Tensor], noise=None) -> Tensor:
    """
    时序示意图 (n_obs_steps=2, horizon=16, n_action_steps=8):

    时间步:        | t-1  | t    | t+1  | ... | t+7  | t+8  | ... | t+14 |
    观察中的帧:    | YES  | YES  | NO   | ... | NO   | NO   | ... | NO   |
    去噪生成动作:  | YES  | YES  | YES  | ... | YES  | YES  | ... | YES  |
    实际执行动作:  | NO   | YES  | YES  | ... | YES  | NO   | ... | NO   |
    """
    # 多相机图像堆叠: (B, C, H, W) → (B, num_cameras, C, H, W)
    if image_features:
        batch = dict(batch)
        batch[OBS_IMAGES] = torch.stack([batch[key] for key in image_features], dim=-4)

    # 将当前观察推入队列
    self._queues = populate_queues(self._queues, batch)

    # 动作队列为空 → 从模型去噪生成新动作
    if len(self._queues[ACTION]) == 0:
        actions = self.predict_action_chunk(batch, noise=noise)  # (B, n_action_steps, action_dim)
        self._queues[ACTION].extend(actions.transpose(0, 1))     # 转置后入队

    return self._queues[ACTION].popleft()

populate_queues 辅助函数 (src/lerobot/policies/utils.py):

def populate_queues(queues, batch):
    for key in batch:
        if key not in queues:
            continue
        for i in range(len(batch[key])):
            queues[key].append(batch[key][i])
    return queues

队列机制：

  1. 每次 selectaction，当前观察被推入对应的观察队列（长度=nobs_steps）
  2. 动作队列为空时 → 从观察队列取最近 nobssteps 帧 → 堆叠 → 去噪生成 horizon=16 步 → 取前 nactionsteps=8 步入队
  3. 每步弹出一个动作执行
  4. 避免每步都跑扩散去噪（每 8 步才跑一次）

4.3 predictactionchunk — 预测

@torch.no_grad()
def predict_action_chunk(self, batch, noise=None) -> Tensor:
    # 从队列堆叠观察历史
    batch = {k: torch.stack(list(self._queues[k]), dim=1) for k in batch if k in self._queues}
    # batch[OBS_STATE]:   (B, n_obs_steps, state_dim)           → (B, 2, 6)
    # batch[OBS_IMAGES]:  (B, n_obs_steps, num_cameras, C, H, W) → (B, 2, 2, 3, H, W)

    actions = self.diffusion.generate_actions(batch, noise=noise)
    return actions  # (B, n_action_steps, action_dim)

4.4 forward — 训练

def forward(self, batch) -> tuple[Tensor, None]:
    if image_features:
        batch = dict(batch)
        for key in image_features:
            if n_obs_steps == 1 and batch[key].ndim == 4:
                batch[key] = batch[key].unsqueeze(1)  # (B, C, H, W) → (B, 1, C, H, W)
        batch[OBS_IMAGES] = torch.stack([batch[key] for key in image_features], dim=-4)

    loss = self.diffusion.compute_loss(batch)
    return loss, None

class DiffusionModel(nn.Module):
    def __init__(self, config: DiffusionConfig):
        super().__init__()
        self.config = config

        # === 计算全局条件维度 ===
        global_cond_dim = robot_state_dim if robot_state else 0

        # === RGB 编码器 ===
        if image_features:
            num_images = len(image_features)
            if use_separate_rgb_encoder_per_camera:
                encoders = [DiffusionRgbEncoder(config) for _ in range(num_images)]
                self.rgb_encoder = nn.ModuleList(encoders)
                global_cond_dim += encoders[0].feature_dim * num_images   # 64 * N
            else:
                self.rgb_encoder = DiffusionRgbEncoder(config)
                global_cond_dim += self.rgb_encoder.feature_dim * num_images  # 64 * N

        if env_state:
            global_cond_dim += env_state_dim

        # === 触觉编码器 ===
        if use_tactile:
            n_sensors = len(tactile_features) if tactile_features else 1
            self.tactile_encoder = TactileTokenEncoder(
                encoder_type=config.tactile_encoder_type,
                input_shape=config.tactile_input_shape,
                feature_dim=config.tactile_feature_dim,   # 64
                n_tokens=config.n_tactile_chunks,         # 1
                dropout=config.tactile_dropout,
            )
            global_cond_dim += n_sensors * n_chunks * tactile_feature_dim

        # === U-Net (1D) ===
        self.unet = DiffusionConditionalUnet1d(config, global_cond_dim=global_cond_dim * config.n_obs_steps)

        # === 噪声调度器 ===
        self.noise_scheduler = _make_noise_scheduler(
            config.noise_scheduler_type,  # "DDPM" 或 "DDIM"
            num_train_timesteps=config.num_train_timesteps,  # 100
            beta_start=config.beta_start, beta_end=config.beta_end,
            beta_schedule=config.beta_schedule,
            clip_sample=config.clip_sample,
            clip_sample_range=config.clip_sample_range,
            prediction_type=config.prediction_type,  # "epsilon"
        )

        self.num_inference_steps = config.num_inference_steps or config.num_train_timesteps

5.2 训练：compute_loss

def compute_loss(self, batch) -> Tensor:
    """
    batch:
      OBS_STATE:    (B, n_obs_steps, state_dim)       如 (16, 2, 6)
      OBS_IMAGES:   (B, n_obs_steps, num_cameras, C, H, W)
      ACTION:       (B, horizon, action_dim)           如 (16, 16, 6)
      action_is_pad: (B, horizon) 填充掩码
    """
    # 第1步: 编码全局条件
    global_cond = self._prepare_global_conditioning(batch)  # (B, global_cond_dim × n_obs_steps)

    # 第2步: 前向扩散 — 采样噪声并加噪
    trajectory = batch[ACTION]                              # (B, horizon, action_dim)
    eps = torch.randn(trajectory.shape)                      # 随机噪声
    timesteps = torch.randint(0, num_train_timesteps, (B,)).long()  # 随机时间步
    noisy_trajectory = self.noise_scheduler.add_noise(trajectory, eps, timesteps)
    # x_t = √(ᾱ_t) × x_0 + √(1-ᾱ_t) × ε

    # 第3步: U-Net 预测
    pred = self.unet(noisy_trajectory, timesteps, global_cond=global_cond)

    # 第4步: MSE 损失
    if prediction_type == "epsilon":
        target = eps               # 预测噪声
    elif prediction_type == "sample":
        target = trajectory        # 预测原始样本

    loss = F.mse_loss(pred, target, reduction="none")
    if do_mask_loss_for_padding:
        loss = loss * ~batch["action_is_pad"].unsqueeze(-1)
    return loss.mean()

5.3 推理：generateactions + conditionalsample

def generate_actions(self, batch, noise=None) -> Tensor:
    # 编码全局条件
    global_cond = self._prepare_global_conditioning(batch)

    # 迭代去噪: 从 (B, horizon, action_dim) 的纯噪声开始
    actions = self.conditional_sample(batch_size, global_cond=global_cond, noise=noise)
    # (B, horizon, action_dim) = (B, 16, 6)

    # 截取 n_action_steps 动作 (从当前观察位置开始)
    start = n_obs_steps - 1       # = 1
    end = start + n_action_steps  # = 9
    actions = actions[:, start:end]  # (B, 8, 6)
    return actions

def conditional_sample(self, batch_size, global_cond, generator, noise):
    """DDPM / DDIM 迭代去噪"""
    sample = noise if noise is not None else torch.randn((batch_size, horizon, action_dim))
    self.noise_scheduler.set_timesteps(self.num_inference_steps)

    for t in self.noise_scheduler.timesteps:
        model_output = self.unet(sample, torch.full((batch_size,), t, dtype=torch.long), global_cond=global_cond)
        sample = self.noise_scheduler.step(model_output, t, sample, generator=generator).prev_sample

    return sample

5.4 全局条件编码：prepareglobal_conditioning

def _prepare_global_conditioning(self, batch) -> Tensor:
    """
    编码所有模态 → 拼接 → 展平为 (B, global_cond_dim × n_obs_steps)
    """
    global_cond_feats = []

    # (1) 机器人状态: (B, n_obs_steps, state_dim) 直接使用
    if robot_state:
        global_cond_feats.append(batch[OBS_STATE])

    # (2) 图像特征: ResNet → SpatialSoftmax → Linear(64)
    if image_features:
        if use_separate_rgb_encoder_per_camera:
            # 每个相机独立编码器
            images_per_camera = einops.rearrange(batch[OBS_IMAGES], "b s n ... -> n (b s) ...")
            img_features_list = torch.cat([enc(imgs) for enc, imgs in zip(rgb_encoder, images_per_camera)])
            img_features = einops.rearrange(img_features_list, "(n b s) ... -> b s (n ...)", b=B, s=n_obs_steps)
        else:
            # 共享编码器
            img_features = self.rgb_encoder(einops.rearrange(batch[OBS_IMAGES], "b s n ... -> (b s n) ..."))
            img_features = einops.rearrange(img_features, "(b s n) ... -> b s (n ...)", b=B, s=n_obs_steps)
        global_cond_feats.append(img_features)  # (B, n_obs_steps, 64 × num_cameras)

    # (3) 环境状态: (B, n_obs_steps, env_dim)
    if env_state:
        global_cond_feats.append(batch[OBS_ENV_STATE])

    # (4) 触觉特征: CNN → flatten → (B, n_obs_steps, n_chunks*feat_dim)
    if use_tactile:
        for tactile_key in tactile_keys:
            tactile_data = batch[tactile_key]        # (B, n_obs_steps, H, W)
            bs, n_obs = tactile_data.shape[:2]
            tactile_flat = tactile_data.flatten(0, 1)  # (B*n_obs, H, W)
            chunks = self.tactile_encoder(tactile_flat)  # (B*n_obs, n_chunks, feat_dim)
            chunks = chunks.flatten(1)                   # (B*n_obs, n_chunks*feat_dim)
            chunks = chunks.view(bs, n_obs, -1)          # (B, n_obs_steps, n_chunks*feat_dim)
            global_cond_feats.append(chunks)

    # 拼接 → 展平时序维度
    return torch.cat(global_cond_feats, dim=-1).flatten(start_dim=1)
    # (B, sum_all_dims × n_obs_steps)

class DiffusionRgbEncoder(nn.Module):
    def __init__(self, config: DiffusionConfig):
        super().__init__()

        # 图像预处理
        if config.resize_shape is not None:
            self.resize = torchvision.transforms.Resize(config.resize_shape)
        if config.crop_shape is not None:
            self.center_crop = torchvision.transforms.CenterCrop(crop_shape)
            self.maybe_random_crop = (
                torchvision.transforms.RandomCrop(crop_shape)
                if config.crop_is_random else self.center_crop
            )

        # ResNet 骨干 (去掉 avgpool + fc)
        backbone_model = getattr(torchvision.models, config.vision_backbone)(
            weights=config.pretrained_backbone_weights
        )
        self.backbone = nn.Sequential(*list(backbone_model.children())[:-2])

        # 可选: BatchNorm → GroupNorm (适合小 batch size)
        if config.use_group_norm:
            self.backbone = _replace_submodules(
                root_module=self.backbone,
                predicate=lambda x: isinstance(x, nn.BatchNorm2d),
                func=lambda x: nn.GroupNorm(num_groups=x.num_features // 16, num_channels=x.num_features),
            )

        # Spatial Softmax 池化 → 32 个关键点
        dummy_shape = compute_feature_map_shape()
        self.pool = SpatialSoftmax(dummy_shape[1:], num_kp=config.spatial_softmax_num_keypoints)  # 32

        # 输出投影
        self.feature_dim = config.spatial_softmax_num_keypoints * 2  # 64
        self.out = nn.Linear(config.spatial_softmax_num_keypoints * 2, self.feature_dim)

    def forward(self, x: Tensor) -> Tensor:
        """x: (B, C, H, W) → (B, 64)"""
        if self.resize: x = self.resize(x)
        if self.do_crop:
            x = self.maybe_random_crop(x) if self.training else self.center_crop(x)
        x = self.backbone(x)    # ResNet → (B, C_feat, H_feat, W_feat)
        x = self.pool(x)        # Spatial Softmax → (B, 32, 2)  32个 (x,y) 关键点
        x = x.flatten(1)       # (B, 64)
        x = self.out(x)         # (B, 64) 最终特征
        return x

Spatial Softmax

Diffusion Policy 的关键视觉特征提取器，将 2D 特征图转换为一组空间关键点坐标：

class SpatialSoftmax(nn.Module):
    """
    输入: (B, C, H, W) 特征图
    输出: (B, K, 2) K 个关键点的 (x,y) 归一化坐标 [-1, 1]

    步骤:
    1. 可选 1x1 Conv 调整通道数
    2. 每个通道做空间 softmax
    3. 计算期望坐标 = softmax_weights @ pos_grid
    """
    def __init__(self, input_shape, num_kp=None):
        self._in_c, self._in_h, self._in_w = input_shape
        if num_kp is not None:
            self.nets = nn.Conv2d(self._in_c, num_kp, kernel_size=1)
        # 预计算归一化坐标网格
        pos_x, pos_y = np.meshgrid(
            np.linspace(-1, 1, self._in_w), np.linspace(-1, 1, self._in_h)
        )
        self.register_buffer("pos_grid", torch.cat([pos_x.reshape(-1,1), pos_y.reshape(-1,1)], dim=1))

    def forward(self, features: Tensor) -> Tensor:
        if self.nets is not None:
            features = self.nets(features)
        features = features.reshape(-1, self._in_h * self._in_w)   # (B*K, H*W)
        attention = F.softmax(features, dim=-1)                     # 通道级 softmax
        expected_xy = attention @ self.pos_grid                    # (B*K, 2) 期望坐标
        return expected_xy.view(-1, self._out_c, 2)                # (B, K, 2)

7. 条件 U-Net 1D：DiffusionConditionalUnet1d

架构：带有 FiLM 条件调制的 1D 卷积 U-Net，定义在 modeling_diffusion.py 后半部分。

│ 解码器 (上采样 + skip connections)           │
│                                             │
│  UpBlock3: Upsample → Conv(2048→1024)       │
│    └── + skip from Down2 → (B, 1024, 4)     │
│                                             │
│  UpBlock2: Upsample → Conv(1024→512)        │
│    └── + skip from Down1 → (B, 512, 8)      │
│                                             │
│  UpBlock1: Upsample → Conv(512→action_dim)  │
│    └── (B, 6, 16)                           │
└─────────────────────────────────────────────┘
                     │
              最终: Conv1d(1×1) → (B, horizon, action_dim)

FiLM 调制细节

# 每个残差块内:
emb = concat([timestep_embedding(t), global_cond])  # (B, 128 + cond_dim)
emb = Linear → Mish → Linear                         # 投影到 scale + bias

if use_film_scale_modulation:
    out = conv(out) * (1 + scale) + bias             # 同时调制 scale 和 bias
else:
    out = conv(out) + bias                           # 仅调制 bias

特征维度变化 (down_dims=(512, 1024, 2048))

输入:     (B,   6, 16)     action_dim=6, horizon=16
Down1:    (B, 512,  8)     stride=2
Down2:    (B,1024,  4)     stride=2
Down3:    (B,2048,  2)     stride=2
Middle:   (B,2048,  2)
Up3:      (B,1024,  4)     skip + conv
Up2:      (B, 512,  8)     skip + conv
Up1:      (B,   6, 16)     最终卷积
输出:     (B,  16,  6)     horizon × action_dim

8. 触觉传感器集成

8.1 共享触觉编码器

文件: src/lerobot/policies/tactile/encoder.py

  • TactileCNN: Conv→BN→ReLU→MaxPool × 3 → Flatten→Linear→ReLU→Dropout→Linear。输入 (B, H, W)，输出 (B, feature_dim)
  • TactileAttentionCNN: 额外的空间注意力门控 + GlobalAvgPool + GlobalMaxPool
  • TactileTokenEncoder: 包装器，可选 Linear(featuredim, ntokens*feature_dim) 投影到多个 token

8.2 DP 中的触觉集成方式

在 DP 中，触觉特征被展平并拼接到全局条件向量中，通过 FiLM 调制影响 U-Net 的每一层：

# 每个传感器: (B, n_obs_steps, H, W)
tactile_data = batch[tactile_key]
bs, n_obs = tactile_data.shape[:2]
tactile_flat = tactile_data.flatten(0, 1)              # (B*n_obs, H, W)
chunks = self.tactile_encoder(tactile_flat)            # (B*n_obs, n_chunks, feat_dim)
chunks = chunks.flatten(1)                             # (B*n_obs, n_chunks*feat_dim)
chunks = chunks.view(bs, n_obs, -1)                    # (B, n_obs, n_chunks*feat_dim)
global_cond_feats.append(chunks)  # 拼入全局条件

与 ACT 的关键区别：

  • ACT：触觉 token 是独立的序列元素，有位置编码，参与 Transformer 自注意力
  • DP：触觉特征被展平为全局条件向量的一部分，通过 FiLM 注入到卷积层的每个残差块

10. 推理流程

输入: 最近 n_obs_steps=2 帧的观察 (从队列堆叠)

1. 图像预处理 (每帧每个相机):
   Resize(144,192) → Crop → ResNet(GroupNorm) → SpatialSoftmax(32 kp) → Linear(64)
   → 每个相机输出 (B, 64)，所有相机+时间步拼接

2. 状态: 直接拼接 (B, n_obs_steps, state_dim) = (B, 2, 6)

3. 触觉: TactileTokenEncoder → flatten → (B, n_obs_steps, n_chunks*feat_dim)

4. 全局条件: concat([图像特征, 状态, 触觉]) → flatten → (B, cond_dim)

5. DDPM 去噪循环 (默认 100 步):
   for t from 100 down to 1:
       noise_pred = UNet(noisy_actions, t_embedding(128), global_cond)
       noisy_actions = scheduler.step(noise_pred, t, noisy_actions).prev_sample

6. 从 horizon=16 步中截取第 2~9 步 (n_obs_steps-1 ~ n_obs_steps-1+n_action_steps)

7. 动作入队 (长度=n_action_steps)，逐步执行

推理特点：

  • 速度较慢：需要 100 步迭代去噪（DDIM 可减少到 ~10 步以加速）
  • 中间频率推理：每 nactionsteps=8 步推理一次
  • 多峰能力：扩散模型天然支持多峰动作分布（如"从左绕"或"从右绕"）
  • 跨 chunk 衔接：通过队列机制平滑过渡，但 chunk 间可能有微小不连续
